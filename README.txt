To run this final project

1. Clone https://github.com/watthell234/Item_Catalog/ repository.

2. Navigate to https://github.com/watthell234/Item_Catalog/tree/master/app

3. Install packages in requirements.txt

4. run models.py to create the database

5. run load_data.py to populate the database

6. run views.py and navigate to localhost:5000 in your browser
